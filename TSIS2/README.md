# TSIS 2: Paint Application — Extended Drawing Tools

Продвинутое приложение для рисования на базе библиотеки **Pygame**. Проект расширяет базовый функционал Paint, добавляя инструменты свободного рисования, заливку и систему сохранения.

## Основной функционал
- ✅ **Pencil & Line**: Свободное рисование кистью и инструмент «Прямая линия» с предпросмотром.
- ✅ **Brush Sizes**: Три уровня толщины (переключение клавишами `1`, `2`, `3`).
- ✅ **Flood Fill**: Инструмент заливки замкнутых областей цветом.
- ✅ **Text Tool**: Динамическая вставка текста (клик для выбора позиции -> ввод -> Enter для подтверждения).
- ✅ **Save Canvas**: Нажмите `Ctrl + S`, чтобы сохранить рисунок в формате `.png` с временной меткой в папку `/saves`.

## Управление
- **Клавиши 1, 2, 3**: Выбор толщины линии.
- **Ctrl + S**: Сохранить текущий холст.
- **Enter**: Подтвердить ввод текста.
- **Escape**: Отменить ввод текста.
