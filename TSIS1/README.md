# TSIS 1: PhoneBook — Extended Contact Management

Данный проект является расширением приложения PhoneBook, разработанного в рамках практических работ 7 и 8. В текущей версии реализована работа с реляционной базой данных, продвинутый поиск и обмен данными в формате JSON.

## Структура проекта
- **data/** — содержит примеры файлов для импорта и экспорта (`contacts.csv`, `contacts.json`).
- **sql/** — SQL-скрипты для создания структуры базы данных и хранимых процедур.
- **src/** — исходный код приложения на Python.

## Основные возможности
1. **Расширенная модель данных**:
   - Поддержка нескольких номеров телефонов для одного контакта (связь 1-ко-многим).
   - Категории контактов (группы: Family, Work, Friend и др.).
   - Поля для Email и даты рождения.
2. **Продвинутый поиск**:
   - Поиск по имени, почте и всем номерам телефонов одновременно через хранимую функцию.
3. **Пагинация**:
   - Интерактивная навигация по списку контактов (Next/Prev) прямо в консоли.
4. **Импорт и экспорт**:
   - Полный экспорт базы в JSON.
   - Импорт из JSON с обработкой дубликатов (проверка по имени).
